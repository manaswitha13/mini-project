# RouteX 🚚

Smart Delivery Optimization System using Graph Algorithms.

## Features

- User Registration & Login
- Secure Password Hashing
- Route Optimization using Dijkstra Algorithm
- Interactive Route Visualization
- Delivery Analytics
- Route History
- Responsive Dashboard
- SQLite Database

## Technology Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap 5
- Graph Algorithms

## Algorithm Used

Dijkstra's Shortest Path Algorithm

## Installation

pip install -r requirements.txt

python app.py
